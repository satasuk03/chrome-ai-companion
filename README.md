# Chrome Companion

A Chrome extension that displays a draggable pixel-art anime companion on every web page. She has emotions, a speech bubble, and a built-in chat panel.

![Manifest V3](https://img.shields.io/badge/Manifest-V3-blue)

## Features

- **Draggable companion** вЂ” click and drag her anywhere on the page
- **Persistent position** вЂ” remembers where you left her across all sites
- **Chat panel** вЂ” click the companion to open a retro-styled chat window
- **Speech bubble** вЂ” auto-shows reactions with a typewriter animation
- **Emotion system** вЂ” 4 sprites: `neutral`, `happy`, `blush`, `angry`
- **Chat commands** вЂ” type `/set-emotion happy` or `/help` in the chat
- **Shadow DOM isolation** вЂ” styles never leak in or out of the host page
- **SPA resilient** вЂ” auto re-injects if the page removes the DOM

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+)
- Google Chrome

### Install & Build

```bash
npm install
npm run build
```

### Load in Chrome

1. Navigate to `chrome://extensions`
2. Enable **Developer mode** (top-right toggle)
3. Click **Load unpacked**
4. Select the project root folder (`chrome-companion/`)
5. Open any website вЂ” the companion appears in the bottom-right corner

### Development

Run the watcher for auto-rebuilds on file changes:

```bash
npm run dev
```

After each rebuild, go to `chrome://extensions` and click the refresh icon on the extension card (or press Ctrl+R on the extensions page).

## Chat Commands

| Command | Description |
|---|---|
| `/set-emotion <name>` | Change sprite вЂ” `neutral`, `happy`, `blush`, `angry` |
| `/help` | List all available commands |

## Message API

Control the companion programmatically from a background script or popup:

```js
// Change emotion
chrome.tabs.sendMessage(tabId, {
  type: 'SET_EMOTION',
  emotion: 'happy'
});

// Show speech bubble with typewriter effect
chrome.tabs.sendMessage(tabId, {
  type: 'SET_BUBBLE_TEXT_ANIMATED',
  text: 'Hello world!',
  speed: 30
});

// Both at once
chrome.tabs.sendMessage(tabId, {
  type: 'SET_EMOTION_AND_TEXT',
  emotion: 'blush',
  text: 'Oh, thank you!'
});
```

## Tech Stack

- **Chrome Extension Manifest V3** вЂ” content scripts, storage API
- **Shadow DOM (closed)** вЂ” full CSS isolation from host pages
- **Pointer Events API** вЂ” reliable cross-browser drag with `setPointerCapture`
- **esbuild** вЂ” sub-10ms bundling, zero config
